# Capstone_Project

## System dependecies
- Ubuntu 20.04
- cmake >= 3.22.3
- g++/gcc >= 9.4.0

## Getting started
```
Git clone https://github.com/peichle/Capstone_Project.git
mkdir build
cd /build
cmake .. && make
./car_detection --input=<link-to-image>
```
